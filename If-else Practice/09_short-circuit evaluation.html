Short-circuit Evaluation

Short-circuit Evaluation in javascript refers to the process of evaluating logical expressions in a way that stop as soon as the result is determined this is commonly used with the logical `&&` (AND) and `||`(OR) operators





Short-circuit Evaluation with `&&` (AND)

For The `&& operator`, expression is evaluated from left to right if any operand is falsy the evaluation stops and returns that falsy value because the entire expression can never be true if at least one operand is false

Short-circuit Evaluation with `||` (OR)

For the `||` operator, the expression is evaluated from left to right. if any operand is truthy,the evaluation stops and returns that truthy value ,because the entire expression is guaranteed to be true if at least one operand is true
