Ternary Operator instead of if-else

The Ternary operator `(condition ? expr1 : expr2)` in JavaScript provides a compact way to express conditional logic compared to traditional 'if-else' statements.
Here are several scenarios where using the Ternary operator might be more suitable than a regular 'if-else' statement:





  1. Assigning Values Conditionally
    2. 
            // Using ternary operator
            const result = (age >= 18) ? "Adult" : "Not Adult";
                    
            // Equivalent to if-else statement
            let result1;
            if (age >= 18) {
                result1 = "Adult";
            } else {
                result1 = "Not Adult";
            }

    Here, the ternary operator succinctly assigns the value of `result` based on the condition (`age >= 18`). It's clear and concise, especially when the logic is straightforward.

  3. Returning Values From Functions
    4. 
            // Using Ternary operators to return a value from a function 
            function GetDiscount(isPremium) {
                return isPremium ? 0.1 :0.05;
            }
    
            //Equivalent to if-else statement
            function GetDiscount(isPremium){
                if(isPremium){
                    return 0.1;
                }
                else{
                    return 0.05;
                }
            }

    When a function's return value depends on a condition, the ternary operator simplifies the implementation and reduces the amount of code.
