Difference Between If - Else Statements

There are alot of Differnce betwwen If else Statements lets discusss them one by one


Feature if else
Purpose :- ued to specify a block of code to be executed if a specified condion is true ued to specify a block of code to be executed if a specified condion is False
Syntax if(condion) {
//code to be executed if condition is true } 		else {
//Code to be executed if condion is false }
Usage example 
                    var age =20;
                    if (age>=18)
                        {
                            document.write("You are Adult");
                        }
                
                    var age =20;
                    if (age>=18)
                        {
                            document.write("You are Adult");
                        }
                        else
                        {
                            document.write("You are not adult");
                        }
Condiotn Requires a condiotn to be evaluated to true for the code block to execute Does not Require condition, executes only if the precending if conditon is false
Code Execution Code inside the ifblock runs if the condiotn is true. Code inside the else block runs if the ifcondtion is false
