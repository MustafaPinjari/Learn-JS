Truthy and Falsy

In JavaScript, the concept of "truthy" and "falsy" values are essential to understanding how conditions are evaluated. these terms refer to how non-boolean values are treated when JavaScript is evaluated expects a boolean such as in conditionals or loops




Falsy Values

A Falsy value is one that is considered false when encountered in a boolean context there are 6 falsy values in javascript

  1. False : The Boolean is false
  2. '0' : The number Zero
  3. '-0' : The number Negative Zero
  4. ` "" ` or ` '' ` or ``` : The empty string
  5. `Null` : Represents the intentional absence of any object value
  6. `undefined` : A Variable that has been declared but not assigned value
  7. `NaN` : Represents Not-a-Number

Any Value that is not falsy is considered Truthy

Truthy Values

A truthy value is one that is considered true when enciuntered in a boolean context

  1. Non-Zero numbers(Both positive and negative)
  2. Non-empty strings
  3. Objects and arrays(even if they are empty)
  4. `true`
