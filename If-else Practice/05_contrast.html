Difference Between If - Else Statements

There are differences between `if` and `else` statements. Let's discuss them one by one.

Feature if else
Purpose Used to specify a block of code to be executed if a specified condition is true. Used to specify a block of code to be executed if the same condition is false.
Syntax if (condition) {
// code to be executed if condition is true
}		 else {
// code to be executed if the same condition is false
}
Usage Example 
var age = 20;
if (age >= 18) {
    document.write("You are an Adult");
}
                
var age = 20;
if (age >= 18) {
    document.write("You are an Adult");
} else {
    document.write("You are not an Adult");
}
Condition Requires a condition to be evaluated to true for the code block to execute. Does not require a condition, executes only if the preceding `if` condition is false.
Code Execution Code inside the `if` block runs if the condition is true. Code inside the `else` block runs if the `if` condition is false.
