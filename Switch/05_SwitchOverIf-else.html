Switch Over if-else statement

In javaScript ,both `break` and `return` serve distinct purpose inside a `switch` statement, but tgey fundamentally affect control flow in different ways :



`break` Statment :

  • Usage : In a `switch` statment, `break` is used to exit the switch block. When Javascript encounters a `break` statement inside a `switch` case, it exits the `switch` statment entirely.
  • Effect : It Prevents fall-through behavior. where without a `break`, execution would contine to the next `case` regardless of whether its condition is met. 

  •                         
                            switch (Expression) 
                            {
                                case value1:
                                    // code to execite when expression === value1
                                    break;
                            
                                case value2:
                                    // code to execite when expression === value2
                                    break;
                                .
                                .
                                .
                            
                                default:
                                    // code to execute if none of the cases match
                                    break;
                            }

`return` Statment :

  • Usage : return is used to end the execution of a function and optionally return a value.
  • Effect : When used inside a switch case, return exits not just the switch block, but also the entire enclosing function where the switch statement is located. 

  •                         
                            function dayOfWeek(day) {
                                switch (day) {
                                  case 1:
                                    return "Sunday";
                                  case 2:
                                    return "Monday";
                                  default:
                                    return "Other day";
                                }
                              }
