Example of if-else if-else Statement in JavaScript

We are going to take an example of if-else if-else statement in JavaScript that checks the value of a variable. In this case, we will determine the category of a person based on their age. 


        
        if (condition) {
            // Block of code to be executed if condition is true
        } else if (condition) {
            // Block of code to be executed if the previous condition is false and this condition is true
        } else {
            // Block of code to be executed if all conditions are false
        }
        
    
        
        function determineAgeCategory(age) {
            if (age < 0) {
                return "Invalid age";
            } else if (age <= 12) {
                return "Child";
            } else if (age <= 19) {
                return "Teenager";
            } else if (age <= 64) {
                return "Adult";
            } else {
                return "Senior";
            }
        }
        
        // Example usage:
        let age = 45;
        document.write(determineAgeCategory(age)); // Output: "Adult"
