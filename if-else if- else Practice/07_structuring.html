Example of if -else if -else statement in javascript

we are going to take an example of if -else if -else statement in javascript to Discuss best practices for structuring complex if-else if-else logic

When dealing with complex `if-else if-else` logic, in JavaScript, it's essential to maintain readability ,scalability, and ease of debbuging.here are some best practices to follow :

  1. Use Proper indentation and formating
    • Ensure Your code is properly indented and formatted .this makes it Easier to read and understand the flow of logic 
      •                 
                    if (condition1) {
                        // Code for condition1
                    } else if (condition2) {
                        // Code for condition2
                    } else if (condition3) {
                        // Code for condition3
                    } else {
                        // Code for none of the above conditions
                    }
  2. Avoid Deep Nesting
    • Deeply nested `if-else` statment can be hard to follow .try to flatten the structure where possible 
      •                 
                    if (condition1) {
                        if (condition2) {
                            if (condition3) {
                                // Code
                            } else {
                                // Code
                            }
                        } else {
                            // Code
                        }
                    } else {
                        // Code
                    }

      Refactored to Avoid Deep Nesting

                      
                    if (condition1 && condition2 && condition3) {
                        // Code
                    } else if (condition1 && condition2) {
                        // Code
                    } else if (condition1) {
                        // Code
                    } else {
                        // Code
                    }
  3. Use Switch Statments for Multiple Conditions
    • When You have multiple conditions based on the same variable, consider using a `switch` statment. 
      •                 
                    switch (variable) {
                        case value1:
                            // Code for value1
                            break;
                        case value2:
                            // Code for value2
                            break;
                        case value3:
                            // Code for value3
                            break;
                        default:
                            // Code for default case
                    }
  4. Extract Logic into functions
    • If the code blocks inside `if-else` statemnets are large ,consider extracting them into seprate functions. this helps in maintaining clean and readable code 
      •                 
                    
                    function handleCondition1() {
                        // Code for condition1
                    }
                    
                    function handleCondition2() {
                        // Code for condition2
                    }
                    
                    if (condition1) {
                        handleCondition1();
                    } else if (condition2) {
                        handleCondition2();
                    } else {
                        // Code for other conditions
                    }
  5. Use ternary Operator for simple Condition
    • For Very Simple conditions ,the ternary Operator can make your code more concise. 
      •                 
                    let result = condition ? valueIfTrue : valueIfFalse;
  6. Use Logical Operator to combine Condition
    • Combine conditions using logical operators (`&&` and `||`) to simplify your `if-else` logic 
      •                 
                    if (condition1 && condition2) {
                        // Code for both conditions true
                    } else if (condition1 || condition3) {
                        // Code for either condition1 or condition3 true
                    } else {
                        // Code for other cases
                    }
  7. Consider Using Objects for Conditional Logic
    • For certain scenarios, using an object to map conditions to corresponding actions can be more scalable and readable. 
      •                 
                    const actions = {
                        condition1: () => {
                            // Code for condition1
                        },
                        condition2: () => {
                            // Code for condition2
                        },
                        condition3: () => {
                            // Code for condition3
                        }
                    };
                    
                    const condition = 'condition1';
                    if (actions[condition]) {
                        actions[condition]();
                    } else {
                        // Default code
                    }
