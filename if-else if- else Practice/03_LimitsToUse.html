Example of if -else if -else statement in javascript

we are going to take an example of if -else if -else statement in javascript to demonstrate how mush time we can use if -else if -else statement in javascript

In JavaScript ,you can chain together as many `else-if` conditions as you need there is no inherent limit to the number of `else if` condition you can use in an `if` statement. Each `else if` condition will be evaluated in sequence ,and once a condition evaluates to `true`,the corresponding block of codes will be executed ,and the rest of the conditions will be ignored
