Syntax of if - else if - else statment

if the conditon if `if` block is true then only the code of `if` block will executed
if it is false then program will execute the code of `else if` block
if it is false then program will execute the code of `else` block 


        
        if(condition)
        {
            //Block of Code to be executed if condion is true in ()
        }
        else if(condition)
        {
            //Block of Code to be executed if condion is true in ()
        }
        else
        {
            //Block of code if condition is false in ()
        }
